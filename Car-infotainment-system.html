This project is an ifotainment system for BMW E30 with 7inches touch screen. it can control different things on car besides deplaying the internal, external temperature and humidity. it can detect the presence of passingers using pressure sensors integrated to the seat and it can automatically open the window analogy depends of external weather in case of detecting high level of smoke inside the car. the windows will be closed when air refreshed.
