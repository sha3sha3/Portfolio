Easy JWT Is a Plugin written in C++ for Unreal Engine. I have developed it while working on my own MMO RPG game project. Lateron i have open-sourced it on github and now it is available of UE marketplace as an optional license. This Plugin provides all the function needed to deal with Json Web Tokens inside Unreal Engine and it is the first on the marketplace.
